The shell file reads all Markdown files with file extension .md and use pandoc to convert to PDF.
Pandoc is a universal document converter and how to install is described at http://www.pandoc.org/installing.html.
Below is the steps needed for installation on MacX
1. brew install pandoc
2. download http://tug.org/cgi-bin/mactex-download/BasicTeX.pkg if PDF is wanted as the output format and install it.
3. open a new terminal to execute the shell file
Done!
